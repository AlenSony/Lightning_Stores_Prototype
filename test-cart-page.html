Cart Page Test Tool

How to use this tool:

  1. First, make sure your development server is running (use npm run dev in the project directory)
  2. Click one of the "Add Test Item" buttons below to populate your cart
  3. Click the "View Cart Page" link to test if the cart is displaying correctly
  4. Use the other buttons to clear the cart or view current cart data
Cart data will appear here when you click "View Current Cart Data"
